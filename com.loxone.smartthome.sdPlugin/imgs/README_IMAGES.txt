IMPORTANT: This plugin requires images to work properly!

You need to create PNG images for:

1. Plugin Icons (in imgs/plugin/):
   - icon.png (72x72px)
   - icon@2x.png (144x144px)
   - category-icon.png (28x28px)
   - category-icon@2x.png (56x56px)

2. For each action folder (switch, dimmer, blind, pulse) in imgs/actions/:
   - icon.png (20x20px)
   - icon@2x.png (40x40px)
   - key.png (72x72px)
   - key@2x.png (144x144px)

3. Additional for dimmer and blind:
   - dial.png (200x200px)
   - dial@2x.png (400x400px)

4. Additional for switch:
   - key_on.png (72x72px)
   - key_on@2x.png (144x144px)

QUICK SOLUTION:
Create simple colored squares as placeholders:
- Switch: Blue
- Dimmer: Yellow
- Blind: Gray
- Pulse: Green

See ../../../IMAGES.md for detailed specifications.

The plugin will NOT appear in Stream Deck until these images are created!
